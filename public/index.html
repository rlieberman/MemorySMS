Give a memory, Get a memory


STEP 1: Pick one of the following prompts to respond to


Describe a memory involving a mirror.
Describe a memory of climbing something.
Describe a memory about a lie you told.
Describe a memory a time you lost something.
Describe a memory of a depressing song.

STEP 2: Enter a memory based on the prompt you chose. It can be as long or as short as you like.
